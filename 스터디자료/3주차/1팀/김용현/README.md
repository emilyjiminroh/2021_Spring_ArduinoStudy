### 느낀점
#### 소프트웨어만 다루는 것과 달리, 아두이노는 직접 눈에 보이기 때문에 색다른 흥미를 느낄 수 있었다.
#### 아두이노 다룰 수 있는 시간이 이번 3주차에는 적어서(핑계) 좀 더 활용해볼 시간이 부족했다.
#### 시간이 된다면, 부저를 활용하여 거리에 따라 음을 조절하여 악기를 만들어 볼 생각이다.
