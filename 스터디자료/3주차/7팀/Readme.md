### 삼색 신호등 만들기
<img src="https://user-images.githubusercontent.com/81242086/114307940-f4992980-9b1c-11eb-8762-448c6f05c1fa.JPG"  width="500" height="400">
미리 해 본 빨간색 LED에 초록색과 노란색을 추가하여 신호등을 만들었다
    
점퍼 케이블도 깔맞춤 해주었다ㅎㅎ
    
<img src="https://user-images.githubusercontent.com/81242086/114308022-522d7600-9b1d-11eb-817b-aced716124f6.JPG"  width="500" height="400">

동영상의 설명대로 LED의 +극에 저항을 꽂아주고 -극은 점퍼 케이블로 브레드 보드의 -에 연결해주었다

주의할 점!! 신호등의 원리에 따라 신호가 동시에 켜지면 안된다
따라서 각각의 LED 다리,저항이 겹치지 않게 꽂아준다

<img src="https://user-images.githubusercontent.com/81242086/114308295-43938e80-9b1e-11eb-8372-fab5eec25a72.PNG"  width="500" height="650">
각각의 LED가 몇 번 핀인지 선언해 준 뒤

녹색,노랑,빨간 순으로 켜지도록 해주었다

노랑 신호는 빨강 전에 잠시 알려주는 신호라서 시간을 더 짧게 잡아보았다!
